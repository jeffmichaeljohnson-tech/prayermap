# Code Review Checklist

Please review the code with the following criteria:

## Correctness
- Are there any logic errors or bugs?
- Are edge cases handled properly?
- Does the code do what it's supposed to do?

## Code Quality
- Is the code readable and well-organized?
- Are variable and function names descriptive?
- Is there appropriate error handling?

## Performance
- Are there any obvious performance issues?
- Are there unnecessary computations or re-renders?
- Is data being fetched efficiently?

## Security
- Are there any security vulnerabilities (XSS, injection, etc.)?
- Is user input properly validated and sanitized?
- Are sensitive data handled securely?

## Best Practices
- Does the code follow project conventions?
- Is there code duplication that should be refactored?
- Are there any anti-patterns?

Please provide specific, actionable feedback with line numbers where applicable.
