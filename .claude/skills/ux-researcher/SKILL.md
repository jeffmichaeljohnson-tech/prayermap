## Journey Map: First Prayer Request

### Stage 1: Discovery
**Actions:** Finds app through friend/search
**Thoughts:** "Will this help me feel less alone?"
**Emotions:** Curious, slightly skeptical
**Touchpoints:** App store, social share, website
**Opportunities:** Clear value proposition, social proof

### Stage 2: Onboarding
**Actions:** Downloads, creates account
**Thoughts:** "This is easier than expected"
**Emotions:** Hopeful, slight anxiety
**Touchpoints:** Onboarding flow, permissions
**Opportunities:** Reduce friction, explain privacy

### Stage 3: First Prayer
**Actions:** Posts first prayer request
**Thoughts:** "Am I doing this right?"
**Emotions:** Vulnerable, hopeful
**Touchpoints:** Compose screen, map view
**Opportunities:** Gentle guidance, examples

### Stage 4: Community Response
**Actions:** Receives prayers from others
**Thoughts:** "People actually care!"
**Emotions:** Surprised, grateful, connected
**Touchpoints:** Notifications, prayer count
**Opportunities:** Show impact, encourage reciprocity

### Stage 5: Returning
**Actions:** Comes back to pray for others
**Thoughts:** "I can help someone too"
**Emotions:** Purposeful, belonging
**Touchpoints:** Map, feed, notifications
**Opportunities:** Easy re-engagement, habit formation

## Research Methods

### User Interviews
- 45-60 minute sessions
- Semi-structured format
- Focus on behaviors, not opinions
- Record and transcribe

### Usability Testing
- 5-7 participants per round
- Task-based scenarios
- Think-aloud protocol
- Screen recording

### Surveys
- Max 10 questions
- Mix of quantitative/qualitative
- CSAT, NPS, custom metrics
- In-app or email delivery

### Analytics Review
- Funnel analysis
- Feature usage heatmaps
- Session recordings
- Cohort behavior

## Key Metrics to Track

| Metric | Definition | Target |
|--------|------------|--------|
| Activation Rate | % completing first prayer | >60% |
| D7 Retention | % returning after 7 days | >40% |
| Prayers per User | Weekly average | >3 |
| Response Rate | % prayers receiving support | >80% |
| NPS | Net Promoter Score | >50 |

## Design Principles for PrayerMap

1. **Sacred Simplicity** - Reduce friction, respect the moment
2. **Warm Community** - Feel supported, never alone
3. **Gentle Guidance** - Help without overwhelming
4. **Trustworthy Privacy** - Earn trust, protect vulnerability
5. **Joyful Celebration** - Highlight answered prayers
