# Code Review

Review the selected code for:

## Quality Checks
- **TypeScript**: Proper typing, no `any`, null safety
- **React Patterns**: Hooks usage, component structure, state management
- **Security**: No exposed secrets, proper RLS, input validation
- **Performance**: Unnecessary re-renders, missing memoization, N+1 queries
- **Error Handling**: Loading/error states, try/catch, user feedback

## Response Format
1. **Issues Found** (if any) - Prioritized by severity
2. **Suggestions** - Improvements that could be made
3. **Positive Notes** - What's done well

Be specific with line numbers and provide corrected code snippets.
